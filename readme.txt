== Description ==

Ermöglicht das Akzeptieren von Terminen vom Frontend und das Verwalten oder Erstellen von Terminen vom Administrator aus.

[HILFE & SUPPORT](https://n3rds.work/intranet/gruppen/psource-communityhub/)

== Screenshots ==

1. Wochenansicht im Frontend
2. Verschiedene Dienstleister wählen
3. Aus verschiedenen Dienstleistungen wählen
4. Anzahlung, Kaution oder vorab den Termin bezahlen lassen
5. PayPal und PSeCommercegateway inkludiert
